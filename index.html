Check GIT version:

    1. git --version"

Initial Setup & Configuration:

Configure your identity across all local repositories

    1. Set username: git config --global user.name "[name]"
    2. Set email: git config --global user.email "[email]"

New local repo:

Starting a Project Create a new repository or clone an existing one.

    1. git init [project-name]
    2. git clone [url]

Check the status of files:

checking the status of untracked files and changes

    1. git status

Stage file and Commit changes :

Commiting the untracked files and changes to local repository

    1. Stage file: git add [filename] Stage all: git add .
    2. Commit changes: git commit -m "[message]"

Push the changes to GitHub Server :

Pushing the files and changes to remote repository on GitHub

    1. git push [remote] [branch]
    2. # Example: git push origin master
